# Commands

`cargo fc` wraps `cargo`: give it a cargo command and it runs that command across your feature matrix. It also adds a handful of its own flags and one subcommand.

- **[Running commands]({{< relref "running-commands.md" >}})** — how arguments are forwarded, package selection, toolchains and aliases.
- **[Output modes]({{< relref "output-modes.md" >}})** — `--diagnostics-only`, `--dedupe`, `--summary-only`, `--show-pruned`, and friends.
- **[The matrix subcommand]({{< relref "matrix.md" >}})** — emit the feature matrix as JSON.
- **[CLI reference]({{< relref "cli-reference.md" >}})** — every flag and subcommand.
